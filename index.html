Multi-Purpose Gas Sensor (MQ-135)

Introduction

The MQ-135 is a gas sensor designed to detect various air pollutants. It is widely used in air quality monitoring, industrial environments, and environmental research.

Key Features

  • Detection Range: Detects ammonia (NH3), NOx, alcohol, benzene, smoke, and CO2.
  • Sensor Type: Solid-state with a heating element and MOX layer.
  • Sensitivity: Resistance varies based on gas concentration.

Technical Specifications

  • Voltage: 5V DC
  • Output: Analog and Digital
  • Heater Resistance: 31Î©
  • Preheating Time: 24-48 hours

Applications

The MQ-135 sensor is used in air purifiers, smart home systems, and environmental monitoring systems.

Sample Arduino Code

#define MQ135_PIN A0 
#define LED_PIN 2 
void setup() { 
    Serial.begin(9600);
    pinMode(LED_PIN, OUTPUT); 
}
void loop() { 
    int sensorValue = analogRead(MQ135_PIN);
    Serial.print("Sensor Value: ");
    Serial.println(sensorValue);
    if (sensorValue < 330) { 
        digitalWrite(LED_PIN, LOW); 
    } else { 
        digitalWrite(LED_PIN, HIGH); 
    }
    delay(1000); 
}
